############################################
Welcome to arrowhead-client's documentation!
############################################

This is the documentation for the ``arrowhead_client`` Python library, which enables you to use Python to design and
test Eclipse-Arrowhead compliant clients with little overhead and trouble.

Click :ref:`eclipse-arrowhead` to learn more about the framework, it's components, and the philosophy behind it,
with some useful links to other resources.

Click :ref:`quickstart` if you just want to get started testing the client library.

Click :ref:`user-guide` to learn how to use the framework, with a focus on how things are used.

Click :ref:`api-reference` for a detailed view of all modules, classes and functions available in this library.
